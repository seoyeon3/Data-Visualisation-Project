# Evolution of Diet and Human Physiological Adaptation
## Project Overview

This project explores the evolution of human diet and its relationship with physiological adaptations over millions of years using biological data of human ancestors.

Through data visualisation, the project investigates how dietary transitions from fruit-based diets to omnivorous and carnivorous patterns are associated with changes in anatomical and functional traits.

## Visualisation Design

The analysis focuses on understanding how diet influences evolutionary adaptations across multiple biological dimensions:

-  **Diet evolution over time**  
  Visualised using a ridge plot to show how different diet types emerged and shifted over millions of years  

-  **Limb usage adaptation (arm & foot types)**  
  Mosaic plots reveal relationships between diet types and functional adaptations such as climbing vs walking 

-  **Cranial capacity & tool use**  
  Box plots demonstrate how brain size increased alongside dietary complexity and tool usage over time 

-  **Tooth enamel thickness**  
  Stacked bar charts show how enamel thickness varies depending on diet hardness and consumption patterns 

-  **Canine shape adaptation**  
  Visual comparisons highlight the relationship between diet (fruit-based vs meat-based) and tooth morphology 

-  **Jaw shape evolution**  
  Analysis of how different jaw structures evolved in response to dietary changes 


## Key Insights

- Human diets evolved progressively from fruit-based to more complex omnivorous and carnivorous patterns over time  
- Dietary changes are strongly associated with functional adaptations, including limb usage (climbing → walking)  
- Increased dietary complexity correlates with larger cranial capacity and the emergence of tool use  
- Harder diets are linked to thicker tooth enamel and distinct dental structures  
- Clear relationships exist between diet type and anatomical features such as canine shape and jaw structure
